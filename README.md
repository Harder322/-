# Telegram-бот для расписания вуза

## Предметная область
Бот предназначен для студентов и преподавателей вуза, чтобы быстро получать актуальное расписание занятий. Пользователь может:
- Указать или сменить номер учебной группы.
- Просматривать расписание на сегодня, завтра, на неделю с выбором дня.
- Управлять уведомлениями о предстоящих парах.
- Сбрасывать сохранённую группу и просматривать информацию о боте.

Бот взаимодействует с базой данных, содержащей информацию о группах, расписании и занятиях.

## Содержимое репозитория
- **diagrams/** — диаграммы UML в формате PlantUML:
  - `usecase.puml` — диаграмма вариантов использования.
  - `class.puml` — диаграмма классов.
  - `sequence.puml` — диаграмма последовательности.
  - `state.puml` — диаграмма состояний.
  - `activity.puml` — диаграмма деятельности.
- **src/** — программный код на Python, сгенерированный на основе диаграммы классов:
  - `models.py` — модели данных (User, Group, Schedule, Lesson).
  - `bot.py` — базовый каркас Telegram-бота.

## Как просмотреть диаграммы
Диаграммы выполнены в синтаксисе PlantUML. Их можно открыть в:
- Плагине PlantUML для IntelliJ IDEA / VS Code.
- Онлайн-редакторе [PlantUML Online Server](https://www.plantuml.com/plantuml/uml/).

## Процесс выполнения задания
1. Определена предметная область.
2. Построены диаграммы, описывающие статическую и динамическую структуру продукта.
3. Из диаграммы классов сгенерирован код на Python, пригодный для дальнейшей разработки.
4. Все артефакты размещены в данном репозитории.
